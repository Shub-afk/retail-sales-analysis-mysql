# Retail Sales Analysis Using MySQL

## Objective

Analyze retail sales data using SQL to identify sales trends, customer behavior, and business opportunities.

---

## Dataset

- Retail Sales Dataset
- Database: retail_sales_db
- Table: retail_sales_dataset

---

## Methodology

1. Imported CSV into MySQL
2. Verified table structure
3. Performed data quality checks
4. Conducted exploratory data analysis
5. Performed sales analysis
6. Analyzed customer behavior
7. Used advanced SQL functions

---

## SQL Concepts Used

- Aggregate Functions
- GROUP BY
- HAVING
- CASE
- Subqueries
- Window Functions

---

## Business Outcomes

- Identified highest revenue category
- Ranked customers by spending
- Compared weekday vs weekend sales
- Measured monthly revenue trends
- Calculated revenue contribution

---

## Conclusion

SQL enables businesses to transform raw transactional data into meaningful insights that support decision-making.
